# sleic-best-practices-2025-12-02

Materials for a presentation Rick Gilmore &amp; Alaina Pearce gave in the SLEIC educational workshop series

The source file is written in [Quarto](https://quarto.org) and rendered using the RevealJS framework to a single HTML file.

The rendered contents are served via GitHub Pages here:

<https://penn-state-open-science.github.io/sleic-best-practices-2025-12-02/>

Use the arrow keys to navigate between the slides.

## Contents

- `index.qmd`, `index.html`: the Quarto source and HTML rendered output files.
- `index_files/`: supporting libraries.
- `include/`: Resources
  -`/img` images,
  - `/bib` references
  - `css/` stylesheets
  - `fig/` code-generated figures

## Usage

To modify or re-render the slides:

1. Clone the repository.
2. Open the project and install the package dependencies as requested by `renv`.
3. Re-render the slides by running `quarto render` in a terminal.
4. Open `index.html` in a browser.
